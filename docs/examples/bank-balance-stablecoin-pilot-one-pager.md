# Permissioned Stablecoin Pilot One-Pager

## Summary

TAP enables a bank or consortium to operate a permissioned stablecoin using private off-chain eligibility checks, policy-linked proof settlement, and maker-checker issuance controls. The result is a more credible tokenized money operating model than simple public-chain issuance with coarse wallet allowlisting.

## What This Solves

- protects customer financial and identity data
- links issuance and transfer controls to deterministic policy state
- preserves consortium control over infrastructure and ecosystem value

## Pilot Scope

- product: permissioned consortium stablecoin
- sources: deposit account balance API and KYC status API
- controls: maker-checker mint approval and transfer compliance proofs
- deployment: self-hosted, private-rollup ready

## Why It Matters

Today, most stablecoin issuance happens publicly. That works for open distribution, but not for institutions that need privacy, auditability, and stronger operational control. TAP demonstrates how a bank can issue tokenized money in a permissioned environment while still keeping an expansion path to public Ethereum bridge rails.

## What the Pilot Demonstrates

- private customer eligibility checks
- policy-linked proof settlement
- auditable issuer governance
- reusable partner integration model for bank APIs

## Deliverables

- configured tenant and provider integrations
- policy version 1 for stablecoin access
- source collect transcripts
- maker-checker mint transcript
- transfer compliance transcript
- redacted demo pack and release bundle

## Expansion Path

- issuer reserve and treasury verification
- consortium multi-issuer stablecoin controls
- private B2B settlement
- bridge conversion into public Ethereum stablecoin rails
