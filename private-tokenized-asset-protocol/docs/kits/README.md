# TAP Kits

TAP is packaged as kits so bank dev teams can fork and run independently.

1. `runtime-kit.md`: core services, APIs, and deployment topology.
2. `integration-kit.md`: partner adapter contracts and onboarding workflow.
3. `operator-kit.md`: reliability, diagnostics, and incident operations.
4. `compliance-kit.md`: policy, audit evidence, and governance controls.
