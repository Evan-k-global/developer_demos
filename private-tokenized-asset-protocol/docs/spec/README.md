# Protocol Spec

This spec is the source of truth for self-hosted bank/consortium deployments of TAP.

Scope:
- TAP is a protocol/runtime for compliant tokenization workflows.
- TAP is not a rollup implementation.
- Private/permissioned sovereign rollup assumptions are documented for integration.
